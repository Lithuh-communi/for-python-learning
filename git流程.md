┌─────────────────────────────────────────────────────────┐
│                     ☀️ 早上到工位                       │
│                                                         │      git clone：从github拉取新项目
│  1. git pull            ← 拉取团队最新代码               │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                     💻 开始干活                         │
│                                                         │
│  2. git checkout -b feature/xxx   ← 创建功能分支         │
│                                                         │
│  3. 写代码...  写代码...  写代码...                       │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                     ✅ 写完了提交                        │
│                                                         │
│  4. git status                ← 看看改了啥               │
│  5. git add .                 ← 暂存所有改动             │
│  6. git commit -m "feat: xxx" ← 提交，写清楚消息          │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                     🔗 合并到主分支                      │
│                                                         │
│  7. git checkout main         ← 切回 main                │
│  8. git pull                  ← 拉最新 main              │
│  9. git merge feature/xxx     ← 合并你的分支             │
│                                                         │
├─────────────────────────────────────────────────────────┤
│                     🚀 推送到远程                       │
│                                                         │
│  10. git push                 ← 推上去就完了！           │
│                                                         │
│  (可选) git branch -d feature/xxx  ← 删掉本地分支         │
